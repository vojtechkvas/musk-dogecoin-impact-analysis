## How to run environment

Create virtual enviroment:

```bash
python -m venv venv
```

Start enviroment:

```bash
source venv/bin/activate
```

Install requiremnts:

```bash
pip install -r requirements.txt
```

Export requiremnts:

```bash
pip freeze > requirements.txt
```

Stop enviroment:

```bash
deactivate
```
